# 🎯 Kick-off: Deploy an AI-enabled web commerce in Azure

Welcome to the **IT/Ops Open Hack Challenge**!  
Your mission: **Solve operational challenges** using **Microsoft Azure native tools**.  

Our partner development team created a new web commerce application. They have deployed it to Azure, and while the application is working, we need to ensure it’s properly secured, monitored, and compliant. Your job is to identify and fix the items that are not compliant. You can use your own experience and creativity to solve the challenges, you can leverate Azure Copilot and Azure AI tools, or you can ask for help from the Microsoft engineers and AI experts.

Here's the application architecture:

⏱ **Duration:** 3 hours  
🎯 **Goal:** Complete all 4 levels to implement best practices and make the web commerce application compliant.

## 🚀 The Challenge

By the end of all 4 levels, you’ll have:

- Secured identity and access for AI services
- Implemented network isolation and private access to AI endpoints
- Set up monitoring, logging, and troubleshooting for AI workloads
- Enforced governance and compliance for AI operations

This is more than just a demo — it’s a crash course in managing real-world AI workloads with Azure.
So let’s go!🔥
